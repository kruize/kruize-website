Kruize Features

Right-sizing

Kruize can provide right-sizing recommendations for containers and namespaces. Container right sizing includes CPU, Memory and Nvidia Accelerators. Namespace recommendations are in the form of limits for CPU and quota for memory.

Box Plot Recommendation support

Visualize cost distributions with detailed statistical insights. Box Plots provides actual usage summary for the observed duration.

GPU Recommendations

MIG Partitioning Recommendations are available from Kruize v0.1 onwards

Instaslice

Kruize uses Instaslice under the covers to apply MIG slicing recommendations for Nvidia accelerators.

Vertical Pod Autoscaler

Kruize uses VPA under the covers to apply CPU and memory right sizing recommendations. Kruize supports two modes “auto” and “recreate” that correspond to the modes of VPA with the same names.
